Materi Pemrograman Menengah

1. Struktur Data Lanjut

Struktur data tingkat lanjut memungkinkan kita untuk mengatur dan mengelola data dengan cara yang lebih efisien dan terstruktur. Di sini, kita akan mempelajari array multidimensi, stack, dan queue.

Array Multidimensi di JavaScript

let matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
];

console.log(matrix[0][0]); // Output: 1
console.log(matrix[1][2]); // Output: 6
console.log(matrix[2][1]); // Output: 8

Array Multidimensi di PHP


$matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
];

echo $matrix[0][0]; // Output: 1
echo $matrix[1][2]; // Output: 6
echo $matrix[2][1]; // Output: 8

Stack di JavaScript (LIFO)

let stack = [];

stack.push(10); // Stack: [10]
stack.push(20); // Stack: [10, 20]
stack.push(30); // Stack: [10, 20, 30]

console.log(stack.pop()); // Output: 30, Stack: [10, 20]
console.log(stack.pop()); // Output: 20, Stack: [10]
console.log(stack.pop()); // Output: 10, Stack: []

Queue di JavaScript (FIFO)

let queue = [];

queue.push(10); // Queue: [10]
queue.push(20); // Queue: [10, 20]
queue.push(30); // Queue: [10, 20, 30]

console.log(queue.shift()); // Output: 10, Queue: [20, 30]
console.log(queue.shift()); // Output: 20, Queue: [30]
console.log(queue.shift()); // Output: 30, Queue: []

2. Fungsi Lanjut

Fungsi tingkat lanjut mencakup konsep seperti rekursif, fungsi higher-order, dan parameter default. Fungsi ini meningkatkan fleksibilitas dan kekuatan kode kita.

Fungsi Rekursif di JavaScript

function faktorial(n) {
  if (n === 0) {
    return 1;
  }
  return n * faktorial(n - 1);
}

console.log(faktorial(5)); // Output: 120

Fungsi Rekursif di PHP


function faktorial($n) {
  if ($n === 0) {
    return 1;
  }
  return $n * faktorial($n - 1);
}

echo faktorial(5); // Output: 120

Fungsi Higher-Order di JavaScript

function operate(a, b, callback) {
  return callback(a, b);
}

function add(x, y) {
  return x + y;
}

console.log(operate(5, 3, add)); // Output: 8

Fungsi Higher-Order di PHP


function operate($a, $b, $callback) {
  return $callback($a, $b);
}

function add($x, $y) {
  return $x + $y;
}

echo operate(5, 3, 'add'); // Output: 8

Fungsi dengan Parameter Default di JavaScript

function greet(name = "Pengguna") {
  console.log("Hello, " + name);
}

greet();      // Output: Hello, Pengguna
greet("Ali"); // Output: Hello, Ali

Fungsi dengan Parameter Default di PHP


function greet($name = "Pengguna") {
  echo "Hello, " . $name;
}

greet();      // Output: Hello, Pengguna
greet("Ali"); // Output: Hello, Ali

3. Penanganan Kesalahan (Error Handling)

Penanganan kesalahan membantu program tetap berjalan meskipun ada error. Di JavaScript dan PHP, kita menggunakan mekanisme seperti `try-catch` untuk menangani kesalahan.

Error Handling di JavaScript

try {
  let angka = 10;
  if (angka > 5) {
    throw "Angka terlalu besar!";
  }
} catch (error) {
  console.log("Terjadi kesalahan: " + error);
}

Error Handling di PHP


  $angka = 10;{
  if ($angka > 5) {
    throw new Exception("Angka terlalu besar!");
  }
} catch (Exception $e) {
  echo "Terjadi kesalahan: " . $e->getMessage();
}

4. Pemrograman Asinkron

Pemrograman asinkron memungkinkan kita untuk menjalankan kode tanpa menunggu kode lainnya selesai, membuat aplikasi lebih responsif dan efisien.

Asynchronous di JavaScript menggunakan setTimeout

console.log("Mulai");
setTimeout(function() {
  console.log("Selesai setelah 2 detik");
}, 2000);
console.log("Berjalan paralel");

Asynchronous di JavaScript menggunakan Fetch

fetch('https://api.github.com/users/octocat')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.log("Terjadi kesalahan: " + error));

Asynchronous di PHP menggunakan sleep()


echo "Mulai\n";
sleep(2); // Delay selama 2 detik
echo "Selesai setelah 2 detik\n";

5. Algoritma dan Kompleksitas Waktu

Algoritma adalah urutan langkah-langkah untuk menyelesaikan masalah, sementara kompleksitas waktu mengukur seberapa efisien suatu algoritma dalam hal waktu yang dibutuhkan berdasarkan ukuran input.

Contoh Algoritma Pencarian Linier (Linear Search)

function linearSearch(arr, target) {
  for (let i = 0; i < arr.length; i++) {
    if (arr[i] === target) {
      return i; // Mengembalikan index jika ditemukan
    }
  }
  return -1; // Tidak ditemukan
}

console.log(linearSearch([5, 3, 8, 4, 2], 8)); // Output: 2

Kompleksitas Waktu Pencarian Linier

Kompleksitas waktu algoritma pencarian linier adalah O(n), yang berarti waktu eksekusi bertambah seiring dengan bertambahnya ukuran array.
