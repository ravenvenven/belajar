Dasar Pemrograman

1. Apa Itu Variabel?

Variabel adalah tempat untuk menyimpan data atau nilai. Nilai ini bisa berubah-ubah selama eksekusi program. Variabel memiliki nama yang unik dan tipe data tertentu.

Contoh Variabel di JavaScript

let nama = "Ali";  // Tipe data String
let umur = 25;     // Tipe data Number
let tinggi = 1.75; // Tipe data Float
let isMahasiswa = true; // Tipe data Boolean

Di atas, kita mendeklarasikan empat variabel dengan berbagai tipe data. Variabel nama adalah string, umur adalah angka, tinggi adalah float, dan isMahasiswa adalah boolean.

Contoh Variabel di PHP


$nama = "Ali";     // String
$umur = 25;        // Integer
$tinggi = 1.75;    // Float
$isMahasiswa = true; // Boolean
echo "Nama: " . $nama . ", Umur: " . $umur;

2. Tipe Data

Setiap variabel memiliki tipe data yang berbeda. Beberapa tipe data dasar yang sering digunakan dalam pemrograman adalah:

  • String: Data berupa teks (misalnya "Halo Dunia")
  • Number: Data berupa angka (misalnya 10, 3.14)
  • Boolean: Data berupa nilai true atau false
  • Array: Kumpulan beberapa nilai dalam satu variabel (misalnya [1, 2, 3])
  • Object: Struktur data yang berisi pasangan key-value (misalnya {name: "Ali", age: 25})

Contoh Tipe Data di JavaScript

let nama = "Ali";      // String
let umur = 25;         // Number
let tinggi = 1.75;     // Float
let isMahasiswa = true; // Boolean
let daftarAngka = [1, 2, 3, 4]; // Array
let person = {name: "Ali", age: 25}; // Object

3. Logika Pemrograman (Pengkondisian)

Logika dalam pemrograman digunakan untuk membuat keputusan. Struktur logika yang paling umum adalah if, else, dan switch.

Contoh Penggunaan if dan else di JavaScript

let angka = 10;

if (angka > 5) {
  console.log("Angka lebih besar dari 5");
} else {
  console.log("Angka kurang dari atau sama dengan 5");
}

Contoh Penggunaan if dan else di PHP


$angka = 10;

if ($angka > 5) {
    echo "Angka lebih besar dari 5";
} else {
    echo "Angka kurang dari atau sama dengan 5";
}

Selain itu, kita juga bisa menggunakan switch untuk menangani beberapa kondisi yang lebih kompleks.

Contoh Penggunaan switch di JavaScript

let hari = "Senin";
switch (hari) {
  case "Senin":
    console.log("Hari pertama minggu");
    break;
  case "Jumat":
    console.log("Hari kerja terakhir");
    break;
  default:
    console.log("Hari biasa");
}

Contoh Penggunaan switch di PHP


$hari = "Senin";
switch ($hari) {
  case "Senin":
    echo "Hari pertama minggu";
    break;
  case "Jumat":
    echo "Hari kerja terakhir";
    break;
  default:
    echo "Hari biasa";
}

1. Apa Itu Array?

Array adalah struktur data yang menyimpan beberapa nilai dalam satu variabel. Setiap elemen dalam array dapat diakses menggunakan indeks yang dimulai dari angka 0.

Contoh Penggunaan Array di JavaScript

let buah = ["Apel", "Pisang", "Mangga"];
console.log(buah[0]); // Output: Apel
console.log(buah[1]); // Output: Pisang
console.log(buah[2]); // Output: Mangga

Contoh Penggunaan Array di PHP


$buah = array("Apel", "Pisang", "Mangga");
echo $buah[0]; // Output: Apel
echo $buah[1]; // Output: Pisang
echo $buah[2]; // Output: Mangga

2. Apa Itu Object?

Object adalah tipe data yang menyimpan data dalam bentuk pasangan key-value. Biasanya digunakan untuk menyimpan informasi yang lebih kompleks.

Contoh Penggunaan Object di JavaScript

let orang = {
nama: "Ali",
umur: 25,
pekerjaan: "Mahasiswa"
};

console.log(orang.nama); // Output: Ali
console.log(orang.umur); // Output: 25
console.log(orang.pekerjaan); // Output: Mahasiswa

Contoh Penggunaan Object di PHP


$orang = (object) [
'nama' => 'Ali',
'umur' => 25,
'pekerjaan' => 'Mahasiswa'
];
echo $orang->nama; // Output: Ali
echo $orang->umur; // Output: 25
echo $orang->pekerjaan; // Output: Mahasiswa

3. Manipulasi String

Manipulasi string adalah proses untuk mengubah, mencari, atau memanipulasi data yang bertipe string (teks). Beberapa fungsi umum untuk manipulasi string termasuk pencarian kata, pemotongan, dan penggabungan string.

Contoh Manipulasi String di JavaScript

let kalimat = "Hello, Dunia!";
console.log(kalimat.length); // Output: 14
console.log(kalimat.toUpperCase()); // Output: HELLO, DUNIA!
console.log(kalimat.substring(0, 5)); // Output: Hello

Contoh Manipulasi String di PHP


$kalimat = "Hello, Dunia!";
echo strlen($kalimat); // Output: 14
echo strtoupper($kalimat); // Output: HELLO, DUNIA!
echo substr($kalimat, 0, 5); // Output: Hello

1. Fungsi Tingkat Lanjut

Fungsi tingkat lanjut mencakup konsep seperti fungsi rekursif, fungsi yang mengembalikan nilai, dan parameter default. Konsep-konsep ini memungkinkan kita untuk membuat fungsi yang lebih fleksibel dan efisien.

Contoh Fungsi Rekursif di JavaScript

Fungsi rekursif adalah fungsi yang memanggil dirinya sendiri untuk memecahkan masalah yang lebih kecil. Contohnya adalah perhitungan faktorial.

  function faktorial(n) {
    if (n === 0) {
      return 1;
    }
    return n * faktorial(n - 1);
  }
  
  console.log(faktorial(5)); // Output: 120

Contoh Fungsi Rekursif di PHP

  
  function faktorial($n) {
    if ($n === 0) {
      return 1;
    }
    return $n * faktorial($n - 1);
  }
  
  echo faktorial(5); // Output: 120

Fungsi dengan Parameter Default di JavaScript

  function greet(name = "Pengguna") {
    console.log("Hello, " + name);
  }
  
  greet();      // Output: Hello, Pengguna
  greet("Ali"); // Output: Hello, Ali

Fungsi dengan Parameter Default di PHP

  
  function greet($name = "Pengguna") {
    echo "Hello, " . $name;
  }
  
  greet();      // Output: Hello, Pengguna
  greet("Ali"); // Output: Hello, Ali

2. Penanganan Kesalahan (Error Handling)

Penanganan kesalahan penting untuk memastikan bahwa program dapat menangani error tanpa terhenti. Di JavaScript dan PHP, kita menggunakan mekanisme seperti try-catch untuk menangani error dengan aman.

Contoh Error Handling di JavaScript

  try {
    let angka = 10;
    if (angka > 5) {
      throw "Angka terlalu besar!";
    }
  } catch (error) {
    console.log("Terjadi kesalahan: " + error);
  }

Contoh Error Handling di PHP

  
  try {
    $angka = 10;
    if ($angka > 5) {
      throw new Exception("Angka terlalu besar!");
    }
  } catch (Exception $e) {
    echo "Terjadi kesalahan: " . $e->getMessage();
  }

3. Pemrograman Asinkron

Pemrograman asinkron memungkinkan kita untuk mengeksekusi kode secara non-blocking, yaitu kode dapat berjalan tanpa harus menunggu proses lainnya selesai. Hal ini sangat penting dalam aplikasi web modern yang memerlukan interaksi real-time dengan server atau basis data.

Contoh Asynchronous di JavaScript menggunakan setTimeout

  console.log("Mulai");
  setTimeout(function() {
    console.log("Selesai setelah 2 detik");
  }, 2000);
  console.log("Berjalan paralel");

Contoh Asynchronous di JavaScript menggunakan Fetch

  fetch('https://api.github.com/users/octocat')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.log("Terjadi kesalahan: " + error));

Contoh Asynchronous di PHP menggunakan sleep()

  
  echo "Mulai\n";
  sleep(2); // Delay selama 2 detik
  echo "Selesai setelah 2 detik\n";

4. Kuis Pemrograman Lanjutan

Cobalah menjawab pertanyaan berikut untuk menguji pemahaman Anda tentang fungsi tingkat lanjut, penanganan kesalahan, dan pemrograman asinkron:

  • Buatlah fungsi rekursif untuk menghitung pangkat (exponentiation) di JavaScript dan PHP!
  • Implementasikan penanganan kesalahan dengan try-catch untuk menangani kesalahan pembagian dengan 0 di JavaScript dan PHP!
  • Buatlah program yang menggunakan pemrograman asinkron untuk mengambil data dari API menggunakan fetch di JavaScript!
